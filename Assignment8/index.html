  • Instructions for using the app

    • You can use the navigation keys to move around the application and press P for the selection of that element. Whereas you can also use the W(up), S(down), A(left) , D(right) for navigation with the nose and use P for the selection

  • Test results

    • Test protocol:
    You have to complete the following five tasks using the calculator but complete with two body parts: (1st with the hands using the arrow keys and P key for pressing, and 2nd with your nose using keys W (for up), A (for left), S (for down), D (for right) and P key for pressing.
    Commands to be performed: 1) (78*345)-9
    2) (4278/2)+56
    3) (35632)-234
    4) (3490*5) + 345
    5) (2*2*2)+(3*3)

    Time using the hand: It took four minutes for participant to complete the commands using one hand.
    Errors using the hand: Participant calculated the last command wrong and was not able to get the right answer. Mainly the reason that contributed to this were speed.
    User experience with one hand: It was easy and straightforward task for me. But using with one hand slowed down my speed since I had to press the P button again and again and that created a mental load on me.
    Time using the nose: It took almost twenty-three minutes to complete the same set of command using the nose.
    Errors using the nose: Identifying when the press button is clicked or not was hard Clicking on wrong navigation buttons repeatedly The whole process was slow and prone to errors
    User experience: The process at the start was very frustrating and it took was very slow for the first command but as I progressed through the command, the process became little fast. My neck was hurt badly at the end of this experience and even my eye side became a little blur in the process because after typing each command I had to look at the screen and then at the keyboard again to find the key. An amazing thing, that happened is that I had to create a mental model before executing the commands by thinking how much I am going to press a particular navigation button.

  • Conclusion
    • After completing this process, I got a very sincere and close sense of what people with the disabilities have to go through when using computers or any general gadget. This assignment has provided me immense motivation to design stuff with an element of universal design.
